# Taskly

**Taskly** — A modern and intuitive task management app designed to help you **organize, track, and complete your daily tasks efficiently**. With a clean interface and powerful features, Taskly makes productivity simple and enjoyable.

---

## Features

- Create, edit, and delete tasks easily
- Mark tasks as completed
- Organize tasks by categories or priorities
- Responsive and clean user interface
- Built with scalability and reliability in mind

---

## Tech Stack

- **MongoDB** – Database for storing tasks and user data  
- **Express.js** – Backend API framework  
- **React.js** – Frontend library for dynamic interfaces  
- **Node.js** – Server runtime  
- **Axios** – HTTP client for frontend-backend communication

# Taskly

**Taskly** — A modern and intuitive task management app designed to help you **organize, track, and complete your daily tasks efficiently**. With a clean interface and powerful features, Taskly makes productivity simple and enjoyable. Supports file attachments and profile pictures using **Blob storage** for seamless cloud integration.

---

## Features

- Create, edit, and delete tasks easily
- Mark tasks as completed
- Attach files to tasks (supports multiple file types)
- Upload and manage profile pictures
- Responsive and clean user interface
- Built with scalability and reliability in mind

---

## Tech Stack

- **MongoDB** – Database for storing tasks, user data, and file metadata  
- **Express.js** – Backend API framework  
- **React.js** – Frontend library for dynamic interfaces  
- **Node.js** – Server runtime  
- **Blob Storage / Vercel** – Cloud storage for file uploads and profile images  
- **Axios** – HTTP client for frontend-backend communication  

---

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/taskly.git
Navigate to the project directory:

cd taskly
Install dependencies for backend and frontend:

# Backend
cd server
npm install

# Frontend
cd ../client
npm install
Create a .env file in the backend folder with your environment variables:

MONGO_URI=your_mongodb_connection_string
BLOB_STORAGE_URL=your_blob_storage_endpoint
BLOB_STORAGE_KEY=your_blob_storage_key
PORT=5000
Start the backend server:

cd server
npm run dev
Start the frontend:

cd client
npm start
Open your browser at http://localhost:3000

Usage
Open Taskly in your browser.

Add new tasks, edit existing ones, or delete completed tasks.

Attach files to tasks or upload a profile picture.

Organize tasks efficiently and track your productivity.
